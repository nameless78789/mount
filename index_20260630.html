Game Utility Tools

펫과 탈것 소환 결과를 확인하고, 보너스 발생 구간과 최적 소환 전략을 분석하는 도구 모음입니다. 사용할 기능을 선택하세요.

Seed

시드 추적기

과거 Seed와 현재 Level/Count를 기준으로 현재 Seed와 증가량을 추적합니다.

Skill

스킬 시드 테이블

스킬 Seed를 기준으로 소환 결과를 테이블 형태로 확인합니다.

Pet

펫 시드 테이블

펫 Seed를 기준으로 등급과 스탯 결과를 테이블 형태로 확인합니다.

Pet Bonus

펫 보너스 최적화

제한된 비용으로 펫 소환 개수를 최대화할 수 있는 선택지를 계산합니다.

Mount

탈것 시드 테이블

탈것 Seed를 기준으로 등급과 스탯 결과를 테이블 형태로 확인합니다.

Mount Bonus

탈것 보너스 최적화

제한된 비용으로 탈것 소환 개수를 최대화할 수 있는 선택지를 계산합니다.

각 도구는 같은 폴더의 JSON 데이터 파일을 사용합니다.
